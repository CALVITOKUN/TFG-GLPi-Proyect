Requisitos Previos

  • Servidor web (Apache, Nginx, etc.)
  • Base de datos (MySQL, MariaDB, PostgreSQL)
  • PHP (versión 7.x recomendada)

Descarga de GLPI

Descarga la última versión de GLPI desde el sitio web oficial o desde un repositorio de código abierto.

  • Descarga

    • Instalación

    1. Descomprime el archivo descargado en el directorio de tu servidor web.
    2. Crea una base de datos para GLPI en tu sistema de gestión de bases de datos.
    3. Configura los permisos de los archivos y carpetas de GLPI según las recomendaciones del manual de instalación.
    4. Accede a la URL donde has instalado GLPI desde un navegador web.
    5. Sigue el asistente de instalación, proporcionando la información solicitada (ubicación de la base de datos, credenciales, etc.).
    6. Una vez completada la instalación, elimina el directorio de instalación (install/) por motivos de seguridad.
